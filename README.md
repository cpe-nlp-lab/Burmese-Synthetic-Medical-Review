# Synthetic Burmese Medical Review Dataset
### Sentiment Classification using Synthetic Data

This repository releases the **first synthetic Burmese (Myanmar) medical review dataset** for sentiment classification research.

This dataset accompanies the paper:

> **Classifying Burmese (Myanmar) Medical Reviews Using Synthetic Data Approach**  
> Hein Minn Thu, Pyae Bhone Moe, Nyein Chan Ko Ko, Aye Hninn Khine  
> Advanced Analytics Lab, Department of Computer Engineering, King Mongkut’s University of Technology Thonburi (KMUTT)

---

##  Overview

Burmese is a **low-resource language** in Natural Language Processing (NLP), especially in the healthcare domain. Due to privacy and ethical constraints, publicly available annotated Burmese medical sentiment datasets are extremely limited.

This project addresses that gap by:

- Translating English medical review datasets into Burmese
- Using both Large Language Models (LLMs) and Machine Translation Models (MTMs)
- Evaluating synthetic data quality via classification experiments
- Conducting human evaluation on translation quality

This dataset enables research in:

- Burmese sentiment analysis
- Low-resource healthcare NLP
- Synthetic data generation
- Translation quality evaluation in medical domain

---

## 📊 Dataset Description

The dataset consists of synthetic Burmese translations derived from two English medical review datasets [Medical Sentiment Analysis using Social Media: Towards building a Patient Assisted System](https://aclanthology.org/L18-1442/): 

### Medical Condition Dataset (MCD)

Labels:
- `Recover`
- `Exist`
- `Deteriorate`

### Medication Review Dataset (MRD)

Labels:
- `Effective`
- `Ineffective`
- `Serious Adverse Effect`

Each subset contains synthetic Burmese translations generated by multiple models.

---

## Synthetic Data Generation

Translations were generated using the following models:

### Large Language Models (LLMs)
- Gemini-2.5-pro
- Gemini-2.5-flash
- Gemini-2.0-flash
- Gemma-3-27b-it
- Gemma-SEA-LION-v4-27B-IT

### Machine Translation Models (MTMs)
- NLLB-200-1.3B
- Google Translate

Translation instruction used: Please translate this text from English to Burmese language.
Do not add any additional information or context. 

## Ethical Considerations

- The dataset contains **synthetic translations only**
- No real Burmese patient data is included
- Original English datasets must be obtained separately
- Synthetic text may contain translation errors or hallucinations

This dataset is released **for research purposes only**.
